## Topic: Bar Charts

### ğŸ“Œ Definition
A **bar chart** is a graphical representation used to display **categorical data**.

Each category is represented by a separate bar.
The height of each bar represents the **frequency (count)** of that category.

Bars do **not touch** because the data is categorical (distinct groups).

---

### ğŸ§  When to Use a Bar Chart

Use a bar chart when:
- The variable is **categorical**
- You want to compare counts between categories

Examples of categorical variables:
- Gender
- Department
- Favorite programming language
- Yes/No responses

Do NOT use a bar chart for continuous numerical data.
